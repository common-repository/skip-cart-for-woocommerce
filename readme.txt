== Description ==

WooCommerce has a flow of product to cart to checkout. 
For many cases, cart page is not necessary and customer can be directly taken to checkout page.
This is specially important in virtual / downloadable products or single product purchase only.
By activating this plugin, woocommerce will skip cart page and will redirect customer to checkout page directly.


== Installation ==

= For automatic installation: =


The simplest way to install is to click on 'Plugins' then 'Add' and type 'Skip cart for WooCommerce' in the search field.


= For manual installation 1: =


1. Login to your website and go to the Plugins section of your admin panel.

2. Click the Add New button.

3. Under Install Plugins, click the Upload link.

4. Select the plugin zip file from your computer then click the Install Now button.

5. You should see a message stating that the plugin was installed successfully.

6. Click the Activate Plugin link.


== Frequently asked questions ==

= Where is a settings page? =

There is no settings page. On activating plugin, it automatically applies redirect to all products.
You can not define per product settings right now. We will be adding it in near future.
